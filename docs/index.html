What is NovaSurvey?

NovaSurvey is a survey analytics platform designed for accessibility. It transforms raw survey data into reliable statistical outputs without requiring deep knowledge of survey methodology.

What You Can Do

📊 Run surveys without statistical expertise
🧹 Clean and repair messy datasets automatically
⚖️ Generate weighted population estimates
📈 Compute uncertainty and variance
🧠 Handle small or sparse datasets
🔐 Protect sensitive information (SDC)

How It Works

1. Create a survey project
2. Import data (CSV / API / database)
3. Run analysis
4. Export clean results, estimates, and reports
