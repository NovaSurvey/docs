Seasonal Adjustment

Performs time-series seasonal adjustment and hierarchical reconciliation (X-13 ARIMA equivalent).

Parameters & Options

REPL Parameter CLI Flag Type Required Description
target --target Column Name Required Value column to adjust.
time_var --time_var Column Name Required Date/time column.
group_var --group_var Column Name Optional Optional series grouping column.
period --period Integer Optional Seasonal period (12=monthly [default], 4=quarterly).
