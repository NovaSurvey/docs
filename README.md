# 🌿 NovaSurvey

**NovaSurvey is a statistical survey engine that lets anyone design, run, and analyze surveys without needing deep knowledge of survey methodology.**

It automates complex statistical procedures such as weighting, imputation, variance estimation, and calibration—so you can focus on your data, not the mathematics behind it.

---

## ✨ What you can do with NovaSurvey

With NovaSurvey, you can:

- 📊 Run surveys and analyze results without statistical expertise  
- 🧹 Automatically clean and repair messy or incomplete data  
- ⚖️ Generate reliable population-level estimates from sample data  
- 📈 Quantify uncertainty with built-in variance estimation  
- 🧠 Handle small or sparse datasets intelligently  
- 🔐 Protect sensitive information using disclosure control  
- 🔁 Reproduce structured, publication-ready survey workflows  

---

## 🚀 Quick Start

A typical NovaSurvey workflow looks like this:

1. Create a project (e.g. *Customer Feedback Survey*)
2. Import survey responses (CSV / database / API)
3. Click **Run Analysis**
4. NovaSurvey automatically:
   - Cleans and validates data
   - Applies statistical adjustments
   - Generates estimates and uncertainty metrics
5. Export results (tables, reports, or processed datasets)

No formulas required.

## 🧩 Core Capabilities

NovaSurvey is built from modular statistical components that operate automatically in the background:

### 📌 Data Processing
- Sampling
- Data editing and validation
- Missing data imputation

### ⚖️ Estimation Engine
- Calibration & weighting
- Variance estimation
- Population-level inference

### 📉 Advanced Analytics
- Seasonal adjustment
- Small Area Estimation (SAE)

### 🔐 Data Protection
- Statistical disclosure control (SDC)

> You do not need to configure these manually—NovaSurvey selects and applies appropriate methods based on your project setup.

## 🎯 Designed for Simplicity

NovaSurvey is designed so that:

> You do NOT need to understand survey methodology to use it effectively.

It is suitable for:
- Data analysts
- Researchers
- Policy teams
- Students
- Applied ML/data workflows

Advanced users can still access full control through configuration, CLI, or API.

## 🖥️ Interfaces

NovaSurvey supports multiple ways to work:

- 💻 **CLI / API (Headless Engine)** – automation and integration for developers  
- 📦 **Portable Projects (.novaproj)** – reproducible survey bundles
